# Code of Conduct

Cypress is a small community project. Keep interactions respectful and on-topic.

## Expected behavior

- Be constructive in code review and issue discussions
- Assume good faith from other contributors
- Stay on topic: issues and PRs are for the project, not general game discussion

## Unacceptable behavior

- Personal attacks, harassment, or discriminatory language
- Deliberate disruption of discussions
- Sharing others' private information without consent

## Enforcement

Violations can be reported via Discord DM to a project maintainer or by emailing the address on any maintainer's GitHub profile. Maintainers may remove comments, close issues, or ban contributors who repeatedly violate this policy.

This applies to all project spaces: GitHub issues, pull requests, and the PvZ FB Modding Discord.
